CSS IDs Vs. Classes

What's the difference

Posted by Ali on 08/10/2015

To the noob, CSS classes and IDs may seem to be interchangeable, but they their main difference lies in specificity. (If you use more than one ID on a page, it will be caught in an HTML validator.)IDs are more specific than classes and should be unique to a specific element. Classes can be used to style many different elements, and, somewhat confusingly, can be used alongside IDs.

Chris Coyer uses a great analogy in this article classes are like bar codes and IDs are like serial numbers. If you need to make a broad sweeping change to the style of a page, you can do so by updating the class, much like updating a price in a bar code will change all of the products with that bar code. Whereas a change to an ID, will affect a specific, unique element on your page only.
