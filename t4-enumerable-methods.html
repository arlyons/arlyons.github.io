Ruby Enumerable Methods

A look at Ruby's group_by method

Posted by Ali on 09/20/2015

Ruby's group_by method transforms an array into a hash with key-value pairs that are dictated by a code block of your choosing. It's an extremely useful method best shown by example.

Take an example of an array of names. Let's say you wanted to create plaques for each name and you were charged by the letter. But, if you can group names of the same size together, you will get a discount since the plaque maker can crank out a bunch of plaques using the same setting (economies of scale and all.)
